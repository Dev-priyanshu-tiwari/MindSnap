# 🧠 MindSnap — Interactive Quiz App

A dark, sleek interactive quiz app built with **HTML, CSS & Vanilla JavaScript**. No libraries. No frameworks. Pure JS magic.

> Lets Play !!

Test NOW -  https://dev-priyanshu-tiwari.github.io/MindSnap/
---

## ✨ Features

- ⚡ **30 Questions** across Tech, Science & General Knowledge
- ⏱️ **Countdown Timer** — animated SVG ring, changes color as time runs out
- 🔥 **Streak Tracker** — consecutive correct answers tracked with live badge
- 🎯 **3 Difficulty Modes** — Easy (20s), Medium (15s), Hard (10s)
- 📂 **4 Categories** — Mixed, Tech, Science, General
- 📊 **Detailed Result Screen** — score ring, accuracy %, avg time, full Q&A breakdown
- 💾 **LocalStorage** — saves best score, best streak, and total games played
- 📱 Fully responsive on all devices

## 🎮 How to Play

1. Choose a **Category** and **Difficulty**
2. Hit **Start Quiz**
3. Answer 10 questions before the timer runs out
4. Build streaks for bonus satisfaction 🔥
5. See your detailed results and try to beat your high score!
